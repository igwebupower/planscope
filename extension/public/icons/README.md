# Extension Icons

Add PNG icons in these sizes:
- icon-16.png (16x16)
- icon-48.png (48x48)
- icon-128.png (128x128)

You can generate these from an SVG using tools like:
- ImageMagick: `convert -background none icon.svg -resize 48x48 icon-48.png`
- Sharp (Node.js)
- Online converters

For development, you can temporarily use any square PNG images.
